WPOrigo Smooth Scrolling Plugin is a very simple, but very useful plugin for WordPress.

== Description ==

This plugin add the smooth scrolling technique to your WordPress site.

== Installation ==

1. Install WPOrigo Smooth Scrolling Plugin either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the WordPress Admin Dashboard

Modify the Scrolldistance:  Admin Dashboard -> WPOrigo SSP -> WPOrigo SSP Options
